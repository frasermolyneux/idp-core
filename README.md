## Documentation

## Overview
Core infrastructure for the Internal Developer Platform (IDP). Deploys shared Azure resources including AI Foundry hub/project, Cosmos DB, Azure Cache for Redis, Azure AI Search, Key Vault, Durable Task Scheduler, User-Assigned Managed Identities, Entra app registrations, and monitoring alerts.

**Tech Stack**: Terraform, Azure (swedencentral), GitHub Actions with OIDC
